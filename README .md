# 3D Solar System Simulation

A 3D interactive solar system simulation built with VPython. All 8 planets orbit the sun with realistic relative speeds, textures, and sizes. Click any planet to zoom in and learn about it.

🌐 **Live Website:** [View Here](https://www.glowscript.org/#/user/akshatkansal/folder/MyPrograms/program/SOLAR-SYSTEM)

## Controls

| Control | Action |
|---|---|
| `Space` | Pause / Resume simulation |
| `Click` on a planet | Zoom into it + show info |
| `Scroll` | Zoom in/out |
| `Right click + drag` | Move camera in 3D |

---

## Features

- All 8 planets with real texture maps
- Saturn's rings
- Relative orbital speeds based on real data
- Click-to-zoom with planet info
- Fully 3D camera movement

## Planet Info

| Planet | Fun Fact |
|---|---|
| Mercury | Smallest planet, closest to the sun |
| Venus | Hottest planet with a poisonous atmosphere |
| Earth | Only planet known to have life |
| Mars | Also known as the Red Planet |
| Jupiter | Biggest planet, made of gas |
| Saturn | Has large rings, most beautiful planet |
| Uranus | Icy giant with retrograde rotation |
| Neptune | Farthest planet from the sun |
